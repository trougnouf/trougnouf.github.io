LED clock project

Material

  • Microcontroller: ATMega328P
    8-20MHz, built-in RTC (needs oscillator), 4.5-5.5V, 28 PDIP, compatible with Arduino libraries.
    $3.50 (taydaelectronics.com)

  • Oscillators
    • 20MHz crystal (MC clock): $0.10 (taydaelectronics) // ATMega328P can only have one oscillator. Try built-in 8MHz first.
    • 32.768kHz crystal (RTC): $0.11 (taydaelectronics)

  • Capacitors
    • 1ÂµF to remove AC from DC source: $0.01 * a handful (taydaelectronics)
    • 22pF *2 around 32.768kHz crystal (12.5 * 10^-12 = c^2 / (2c) + 1.3 * 10^-12 -> C = 2.24 E -11 = 22.4pF): $0.01*2 (taydaelectronics)
    • // 27pF *2 around 20MHz crystal (18.5E-12 = c^2 / (2c) + 5E-12 -> C = 27pF): $0.01*2 (taydaelectronics)

  • LEDs:
    time: 8x10 px per num -> 16x30 pixels (RGB)
    date: 8x10x2 (white)
    8x10x6 = 640 leds => 160 red (hours)@$0.02, 160 green (minutes)@$0.03, 160 blue (seconds)@$0.05, 160 white (date)@$0.05
    day#(panel7): 6x10, weekday(panel8): 10x10

  • Shift registers
    • 2*8*5*8 -> 2+5 74HC595. may be able to use unlatching ones on one side, investigate.

  • Resistors
    • LEDs: i=.020A, Vr=2.2V, {Vg,Vb,Vw}=3.4V, Vs=5V
      V=iR; R=V/i; Rr=(5-2.2)/.020=140ohms (*8) (get 150ohms), {Rg, Rb, Rw}=(5-3.4)/.020=80ohms (*24) (get 82ohms)
    • Transistors: 10K * 48

  • Stripboards: $1.29*6,?
  • Transistors: SR*8=6*8=48
